# CMDB
这是我的自动化运维学习项目

## CMDB能做什么
 * 记录资产信息
 * 对资产信息进行增删改查
 * 用户管理
 * 权限控制
 * 定时自动更新资产信息

## 开发语言与框架：
 * 编程语言：Python3.6 + HTML + jQuery
 * 前端Web框架：Bootstrap
 * 后端Web框架：Django
 * 后端Task框架：Celery + Redis
 * 消息队列: Rabbitmq

## CMDB环境要求
 * 编程语言：Python 3.5
 * 操作系统：CentOS 6+
 * Ansible版本：2.5.0 
 * Celery：4.1.0
 * django-celery-beat 1.1.0
